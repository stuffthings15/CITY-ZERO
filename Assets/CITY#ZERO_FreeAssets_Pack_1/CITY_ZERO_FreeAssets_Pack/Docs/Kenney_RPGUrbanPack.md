# Kenney RPG Urban Pack (CC0)

This document summarizes the **Kenney RPG Urban Pack**, a free pixel‑art tileset
created by Kenney.nl for use in top‑down games.  The pack contains more than
**480** 32×32‑pixel tiles for constructing city streets, buildings, vehicles
and other urban props.  It also includes **six characters** with walk
animations and comes with both spaced and non‑spaced tilemaps as well as
separate PNG tiles and import instructions【884328534695308†L125-L143】.

## Contents

- **Tiles:**  A large variety of road segments, intersections, sidewalks,
  buildings, shops, street furniture and vehicles.  The tiles use a
  consistent 32×32 pixel grid and can be mixed and matched to build dense
  urban environments.
- **Characters:**  Six animated characters with directional walk cycles
  suited for the overhead perspective.
- **Tilemaps:**  Ready‑made tilemap images with and without spacing for
  quick import into various engines.
- **Separate PNGs:**  Each tile is also provided as an individual PNG file,
  making it easy to assemble custom tilesets.
- **Documentation:**  Simple import instructions for popular game engines.

## Usage

To use the tileset in your own projects, extract the ZIP file found in the
`Assets` folder and import the tilemaps or individual tiles into your
game engine.  Because the assets are CC0, you may remix or redistribute
them; however, it is polite to mention Kenney.nl in your credits if you
find the pack useful.
