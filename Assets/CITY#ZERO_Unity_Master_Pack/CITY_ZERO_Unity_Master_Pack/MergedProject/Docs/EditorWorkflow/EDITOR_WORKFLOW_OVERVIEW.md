# Editor Workflow Overview

This pack is intended to reduce manual setup time inside Unity.

Included:
- Mission authoring window for generating starter mission JSON files
- Config stub generator window for district, faction, vehicle, weapon, shop, event, and mission JSON
- Scene validation window for checking common reference/setup issues
- Custom inspectors for mission triggers, world markers, and bootstrap flow scripts
- Quick menu items for opening common tools

These are starter editor tools, not a full proprietary pipeline.
