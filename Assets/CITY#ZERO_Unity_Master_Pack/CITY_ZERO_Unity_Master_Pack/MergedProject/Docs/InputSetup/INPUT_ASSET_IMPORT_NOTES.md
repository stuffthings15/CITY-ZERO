# Input Asset Import Notes

File included:
- `Assets/_Project/Data/Input/CITY_ZERO.inputactions`

In Unity:
1. Import the file into the project
2. Select it and allow Unity Input System to generate the asset import data
3. Optionally generate a C# wrapper from the inspector
4. Bind action references to:
   - `PlayerController`
   - `VehicleDriverInputRelay`
   - `InputSystemUIInputModule`

This file is a strong starter but may still need minor edits depending on Unity package version.
