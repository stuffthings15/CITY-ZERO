# Player Input Actions

Create one Input Actions asset named `CITY_ZERO.inputactions`.

## Action Maps

### Player
- Move (Value, Vector2)
- Look (Value, Vector2)
- Sprint (Button)
- Roll (Button)
- Interact (Button)
- Fire (Button)
- Reload (Button)
- Pause (Button)

### Vehicle
- Move (Value, Vector2)
- Handbrake (Button)
- ExitVehicle (Button)

### UI
- Navigate (Value, Vector2)
- Submit (Button)
- Cancel (Button)
- Point (Value, Vector2)
- Click (Button)

## Suggested Bindings

### Keyboard + Mouse
- Move: WASD
- Look: Mouse Position
- Sprint: Left Shift
- Roll: Space
- Interact: E
- Fire: Left Mouse
- Reload: R
- Pause: Escape
- Vehicle Handbrake: Space
- Exit Vehicle: E

### Controller
- Move: Left Stick
- Look: Right Stick
- Sprint: Left Stick Press or Right Trigger modifier
- Roll: South Button
- Interact: West Button
- Fire: Right Trigger
- Reload: X or West
- Pause: Menu
- Vehicle Handbrake: South Button
- Exit Vehicle: West Button
