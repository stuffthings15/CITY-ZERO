# Mission 03 — Dead Drop Dive

**Giver:** Ivy Quill  
**District:** Old Quarter rooftops  
**Type:** Courier / pursuit escape  
**Purpose:** Teach on-foot navigation, rooftop routes, low-heat escape

## Objectives
1. Pick up package from dead-drop terminal.
2. Run package through rooftops and fire escapes.
3. Reach rooftop mailbox.
4. Lose any attention before payout.

## Complications
- Rival lookout attempts interception.
- Package glows on enemy minimap after 90 seconds.

## Rewards
- Cash 900
- Contact unlock: Nox Vale
- Velvet Circuit rep +4
