# Next Pack Overview

This add-on is designed to accelerate the first playable vertical slice.

Included systems:
- AI agents for civilians, police, and faction members
- Mission objective handlers for pickup, delivery, escape heat, destroy target, hold zone
- Mission factory for auto-building mission runtimes from config
- Save manager and JSON save system
- In-game debug menu and console commands
- Vehicle spawner and lightweight traffic lane follower
- Simple inventory and runtime lookup databases

Use this package as a production starter, not a final tuned implementation.
