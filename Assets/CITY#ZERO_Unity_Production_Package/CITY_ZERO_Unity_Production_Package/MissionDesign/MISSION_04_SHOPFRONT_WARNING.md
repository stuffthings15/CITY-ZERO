# Mission 04 — Shopfront Warning

**Giver:** Mara Kells  
**District:** Old Quarter commercial strip  
**Type:** Intimidation / melee  
**Purpose:** Teach melee, crowd reaction, collateral consequences

## Objectives
1. Visit three shops refusing protection payments.
2. Smash tagged property outside each location.
3. Avoid killing civilians.
4. Return to Mara.

## Complications
- One shop owner is armed with a shotgun.
- Civilian panic can trigger police reports.

## Scoring Modifiers
- Reduced payout for civilian injury.
- Bonus if completed with no firearms used.

## Rewards
- Cash 700
- Blue Saints rep +10
