# CITY//ZERO Unity Production Package

This export contains the Unity-specific production package split into properly named files.

## Folder Overview
- `Docs/` — architecture and implementation docs
- `DataSchemas/` — JSON schema examples
- `MissionDesign/` — scripted mission files
- `Balancing/` — handling and tuning tables
- `Production/` — vertical slice and execution planning
- `UnityProjectStructure/` — Unity folder structure and conventions

## Recommended Import Order
1. Read `Docs/UNITY_TECHNICAL_OVERVIEW.md`
2. Review `UnityProjectStructure/UNITY_PROJECT_STRUCTURE.md`
3. Implement schemas from `DataSchemas/`
4. Build missions from `MissionDesign/`
5. Tune vehicles from `Balancing/VEHICLE_HANDLING_TABLES.md`
6. Scope the first build with `Production/VERTICAL_SLICE_CHECKLIST.md`
