# Hand‑Drawn Square Characters (8‑Direction)

This small character pack contains four hand‑drawn **square characters** designed for
top‑down games.  Each sprite faces **eight directions** and is supplied at a
resolution of **128×128 pixels** with each pose and animation separated into
individual PNG files.  The pack provides idle/walk cycles, jump animations and
explosive death effects along with a simple gun sprite【24387402072256†L140-L156】.

## Contents

- **Characters:** base character, hero, skeleton and monster【24387402072256†L151-L156】.  
- **Animations:** idle/walk loops, jumps and a death/explosion effect【24387402072256†L151-L156】.  
- **Sprites:** eight directional views for each character and separated PNG files
  for ease of import【24387402072256†L140-L148】.  
- **Resolution:** 128×128 pixels per frame【24387402072256†L140-L144】.
